Augment Code Toolkit v1.4.0

企业级代码开发工具套件 - 重大版本更新

版本更新概述

  • 编辑器支持 - 新增 IntelliJ IDEA 支持，现已全面支持 Visual Studio Code、Cursor、IntelliJ IDEA 三大主流编辑器
  • 版本兼容 - 完全支持 Augment v0.498.0 最新版本，智能版本适配机制
  • 策略优化 - 针对不同编辑器采用最优重置策略，有效保留用户个人偏好设置
  • 体验升级 - 修复监控功能卡顿问题，新增停止功能，显著提升响应流畅度
  • 网络优化 - 采用 IP1 优先连接策略，50% 概率选择最优网络节点

核心功能模块

  • 重置工具 - 支持 Visual Studio Code、Cursor、IntelliJ IDEA 三大编辑器的智能重置
  • 网络环境 - 搭建安全稳定的网络连接环境，IP1 优先策略，智能检测规避
  • 邮箱服务 - 一键生成临时邮箱地址，自动识别和处理验证码
  • 邮件监控 - 实时监控邮件状态，支持随时停止操作，音效提醒功能

编辑器支持详情

  • IntelliJ IDEA 新增 - 智能重置策略，完整保留插件配置和用户偏好设置
  • Cursor 编辑器 - 完整支持 Cursor 编辑器的重置和配置管理
  • Visual Studio Code - 支持标准版和 Insiders 预览版的完整功能

界面体验优化

  • 选择界面 - 直观的编辑器选择对话框，图标化界面设计，一键选择目标编辑器
  • 用户描述 - 去除复杂技术术语，采用用户友好的描述方式，清晰说明操作影响
  • 状态反馈 - 智能按钮状态管理，监控过程中可随时停止，状态反馈更加清晰
